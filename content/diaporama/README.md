# Photos du diaporama

Déposez ici les photos à afficher dans le **diaporama pleine largeur** de la
page. Toutes les images de ce dossier défilent automatiquement, les unes à côté
des autres.

- **Formats acceptés** : `.jpg`, `.jpeg`, `.png`, `.webp`
- **Ordre d'affichage** : par ordre alphabétique du nom de fichier.
  Pour choisir l'ordre, préfixez les noms par des numéros :
  `01-entrainement.jpg`, `02-competition.jpg`, etc.
- **Ajouter une photo** : copiez le fichier dans ce dossier.
- **Retirer une photo** : supprimez le fichier.

(Après modification, le site doit être reconstruit/publié pour voir les
changements.)
